# Stimulus

Create or update a stimulus on the service provider with HTML content and optional configuration

## Example Usage

```typescript
import { Stimulus } from "qti";

let value: Stimulus = {
  identifier: "<value>",
  title: "<value>",
  content: "<value>",
};
```
