# TestPart

Complete test part structure as parsed from XML.

## Example Usage

```typescript
import { TestPart } from "qti";

let value: TestPart = {
  identifier: "<value>",
  navigationMode: "nonlinear",
  submissionMode: "individual",
  qtiAssessmentSection: [
    {
      identifier: "<value>",
      title: "<value>",
    },
  ],
};
```
